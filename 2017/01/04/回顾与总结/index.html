回顾与总结

发表于 2017-01-04   |  

2016年转眼就过去了，在这2017到来的时候，突然想写点东西回顾下，主要还是想以后来看看。那么开始了….

（一）、Look Back

Work Experience
  1. 爱邦保险app第一版。3月份开始上手项目，项目设计、需求等都比较明确，开发比较顺利。5月份左右结束开发。
  2. 东汇ETC版本迭代。此版本需求变化大改，需要重构，在6月份左右上手项目，东汇ETC项目是一款停车类软件，软件与硬件结合，“物联网”概念。功能完成后测试难度较大，因为需要等测试去停车场反馈问题再来改，有时候场景复现也不易，所以在开发时候逻辑思考要比较全面。项目9月份底结束，已经完成2.1.0版本。
  3. 微信端东汇养车项目参与少量功能开发。乘此机会学习了下前端内容，h5相关内容，javascript等。
  4. 公司12月份申请了微信小程序开发账号，学习了下小程序开发语言（其实底层就是h5和javascript再一次包装），开发工具</>更新较快，基本UI搭建和数据请求可以完成，但是由于小程序的局限性（不知道入口在哪等等，学习结束，听说1月9号将会出来，期待…）。
  5. 学习github上开源库。有很多意外收获。感谢开源！！！

（二）、Look Forward

NewYear Plan
  1. 学习当下比较流行的一门前端语言，RN，Node.js等，这些前端语言发展的真是很快，跨平台简直很有吸引力。”write once,run everywhere”。
  2. 向开源进击自己的力量。
  3. 看更多的书。16年目测看完了5本书，17年争取比这多。
  4. 多出去走走，太宅了。。。
