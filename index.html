Troll Store is a permanently signed, non-jailbroken App that allows users to install unsigned PA files on iPhone without jailbreaking. The principle is to exploit a certificate signature vulnerability called AMF1/CoreTrust to avoid verifying whether the system root certificate is legitimate. It is currently known to exist in iOS 14.0 – iOS 15.4.1 versions.
